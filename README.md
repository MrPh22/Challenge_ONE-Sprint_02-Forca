# Challenge ONE - Iniciante em Programação - Sprint 02 - Jogo da Forca 


**Projeto de um jogo da forca para o desafio de número 2 da trilha de conhecimento "Iniciante em Programação" do Programa ONE + Alura. 
Coloncando em prática todo o conhecimento adquirido na trilha**

![image](https://user-images.githubusercontent.com/117610406/215581179-57786b08-84e1-49a1-825e-d1561e78277c.png)

## Uma Nova Partida

Como todo jogo da forca, quando for iniciada a partida, será colocada aleatoriamente uma palavra para que o jogador tente acertá-la digitando uma letra por vez.
<br>

Neste jogo o usuário terá 8 chances para colocar as letras corretas. A cada erro, a forca começará a ser desenhada e as letras erradas aparecerão abaixo dela.

## Adicione Suas Próprias Palavras

Na página inicial, há outro botão (além do "Iniciar Jogo"), chamado "Adicionar Palavra". O qual o jogador verá uma página nova onde poderá acdicionar qualquer
palavra ao jogo. Porém com um ressalva, o programa só aceitará palavras com, no máximo, 8 letras.
<br>
<p align="center">
  <img src="https://user-images.githubusercontent.com/117610406/215594319-1ff149eb-3a2b-407e-bc7e-104fea0ea7e0.png">
</p>
<p align="center">
  <img width="720" height="480" src="https://user-images.githubusercontent.com/117610406/215596492-934acd75-9f3c-4664-80ff-05dacd3c73d5.gif">
</p>
Para que o usuário salve a palavara e consiga jogar com ela em uma partida futura, ele deverá pressionar o botão "Salvar e começar", que colocará o que foi digitado
junto com as outras palavras preestabelecidas, para que seja sorteada em alguma partida. Além disso, pressionado o botão, um novo jogo começará. Mas se o usuário mudar
de ideia e querer sair dessa página, basta apertar o botão "Cancelar", que o levará à página inicial.
<br>

## Propriedades do Programa

**Todos os botões terão interações quando colocar o cursor do mouse sobre eles.**
<br>
**Se o jogador pressionar qualquer outra tecla do teclado, que não seja uma letra, o programa irá reconhcer e não contabilizará como erro ou acerto na forca ou na
aba de acdicionar uma nova palavra, não será registrada.**
<br>
**O usuário poderá digitar as letras tanto em minúsculas como em maiúsculas, pois o programa irá reconhecer e mostra a ele sempre em maiúculo.**
